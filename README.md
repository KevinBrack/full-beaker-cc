# Pixabay API Search

## ENVIRONMENT
This project requires an API key to reproduce. Create a `.env` file in the root directory and set the key with the following var name.

```
REACT_APP_PIXABAY_KEY=
```

## Tech Stack

 - React Hooks
 - Context
 - Sass
 - Zeit Now Deployment
